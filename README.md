# ШГАБ - Хоригдогчийн мэдээллийн систем

Хоригдогчийн бие даан мэдээлэл авах портал. Хоригдол 6 оронтой нэг удаагийн кодоор нэвтэрч өөрийн мэдээллийг уншиж танилцах боломжтой.

## Технологийн стек

- **Node.js:** v20
- **Frontend:** Next.js 16, React 19, TypeScript
- **UI:** Ant Design 5, Custom CSS
- **Backend:** NestJS (Нэг backend дээр ажиллана)
- **Баазын сан:** Oracle Database

## Боломжууд

| Хэсэг | Тайлбар |
|---|---|
| **Ялын тооцоо** | Шийтгэх тогтоолын огноо, хорих хугацаа, шагналын хоног, ял дуусах огноо |
| **Данс** | Хоригдогчийн дансны мэдээлэл |
| **Зөрчил** | Зөрчлийн бүртгэл |
| **Эрүүл мэнд** | Архаг өвчин, эмийн хэрэглээ, харшил |
| **Уулзалт** | Сүүлийн уулзалтын мэдээлэл |
| **Өргөдөл гомдол** | Хоригдогчийн гомдлын бүртгэл |

## Суулгалт

```bash
npm install
```

## Ажиллуулах

```bash
# Development серверийг 3001 порт дээр ажиллуулна
npm run dev
```

Хөтөч дээр http://localhost:3001 нээнэ.

## Тохиргоо

Портал нь backend серверийн `/prisoner-code-auth/` endpoint-уудтай ажиллана.

## Лавлагаа

- Хоригдогчийн код: 6 оронтой тоо (жишээ: 804822)
- Нэг удаагийн код → JWT token → Dashboard
- Зөвхөн уншигдах (read-only) горимд ажиллана
